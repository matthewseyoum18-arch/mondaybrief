# MondayBrief

Public portfolio proof for a GTM engineering system.

MondayBrief turns public city signals into a weekly outbound workflow for local
B2B services. It ingests permit/license data, matches opportunities against a
customer's existing book of business, scores lead fit, generates a sales brief,
and supports recurring delivery plus feedback.

The application code lives in [`mondaybrief/`](mondaybrief/).

## Why this is relevant

This repo shows the kind of system I would build in a GTM engineering role:

- source discovery and ingestion
- enrichment and scoring
- account prioritization
- sales-ready messaging
- workflow automation
- feedback loops for improving targeting

For Surface Labs specifically, the same pattern maps to a GTM system like Lead
Leak Radar: find companies visibly leaking demo demand, produce evidence-backed
mini-audits, and route the best accounts into targeted outbound.
